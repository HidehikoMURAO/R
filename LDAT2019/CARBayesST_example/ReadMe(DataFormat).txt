

疫病地図(disease mapping)を作成する計算コードとデータ
code.R  : コード

dat.csv   : 街区別のデータ
・IG      : 街区のID
・year    : 年次
・observed: 被説明変数(呼吸器疾患の患者数)
・expected: オフセット(年齢構成・性比等から期待される患者数(offset)
・pm10    : 説明変数1(PM10の濃度)
・jsa     : 説明変数2(失業率)
・price   : 説明変数2(平均不動産価格)

zones.shp: 街区のポリゴンデータ
・IG      : 街区のID
・name    : 街区名（不使用）
・easting : 重心点のX座標(不使用)
・northing: 重心点のX座標(不使用)


フォーマット
・時間順(第1優先),ゾーンID順(IG; 第2優先)でデータを並べておく
・各時点のゾーンは同じでなくてはならない
・被説明変数にNA(欠損値)があってもよい
