
時空間補間を行う計算コードとデータ
code.R  : コード(csvから読み込む場合)
code.R  : コード(R上でsimulationデータを生成して正解と比較する場合)


dat1.csv: 観測地点のsimulationデータ (遷移カーネルは全時点共通と仮定して生成)
・time  : 時点
・s1    : X座標
・s2    : Y座標
・z     : 被説明変数(simulation data)


dat2.csv: 観測地点のsimulationデータ (遷移カーネルが場所毎に異なると仮定して生成)
・time  : 時点
・s1    : X座標
・s2    : Y座標
・z     : 被説明変数(simulation data)


フォーマット
・時間順(第1優先),場所順(第2優先)でデータを並べておく
・各時点の観測地点は同じでなくてはならない
・欠損地点のデータは不要
